## ğŸ“‹ Platform-Specific Details

### **macOS (Mac)**

**How it works:**
1. Installs **Homebrew** (Mac package manager) if not present
2. Uses Homebrew to install Python, Node.js, and Chrome
3. Runs the Python installer for app dependencies

**What you'll see:**
- "Installing Homebrew..." (if needed)
- May ask for your **password** (normal for Mac software installation)
- "Installing Python 3..." (if needed)
- "Installing Node.js..." (if needed)
- "Installing Google Chrome..." (if needed)
- "Installing Python Dependencies..."
- "Installation Completed Successfully!"

**Time:** 5-10 minutes (first time)

---

### **Windows**

**How it works:**
1. Detects if you have **winget** (Windows Package Manager)
2. If you have winget: Automatically installs everything
3. If no winget: Provides download links and guides you

**What you'll see:**
- May ask to **"Run as Administrator"** (for automatic installation)
- "Installing Python 3..." (if needed)
- "Installing Node.js..." (if needed)
- "Installing Google Chrome..." (if needed)
- "Installing Python Dependencies..."
- "Installation Completed Successfully!"

**Time:** 5-10 minutes (first time)

**Note:** Windows 10/11 usually has winget. Older versions may need manual installation.

---

### **Linux**

**How it works:**
1. Detects your Linux distribution (Ubuntu, Fedora, Arch, etc.)
2. Uses appropriate package manager (apt, dnf, pacman)
3. Installs everything automatically

**What you'll see:**
- Will ask for your **sudo password** (normal for installing software)
- "Detected: Ubuntu..." (or your distro)
- "Installing Python 3..." (if needed)
- "Installing Node.js..." (if needed)
- "Installing Google Chrome..." (if needed)
- "Installing Python Dependencies..."
- "Installation Completed Successfully!"

**Time:** 5-10 minutes (first time)

**Supported Distributions:**
- Ubuntu, Debian, Linux Mint
- Fedora, RHEL, CentOS
- Arch, Manjaro
